介绍什么是递归

Before we discuss the recursion, we need to import some concepts.

The closed—form expression is a mathematical expression that can be evaluated in a finite number of basic operations. For example

So what is the recursion?

Take the sequence as an example.

Firstly, we need a simple base case such as $a_1=1,a_2=1$

Secondly, we need a recursive step —a set of rules thet reduces all other cases toward the base case.Just like $a_n=a_n-1+a_n-2$

Then we got a recursion formula for the sequence. Obiviously ,we can get the $a_n$ easily by calculating the $a_{n-1},a_{n-2},a_{n-3}…etc$

In fact, what we define is a very famous formula called Fibonacci sequence. I believe everyone present here can write a program to calculate the $a_n$ whose time complexity is O(n)

However, we still don’t know the closed-form expression of the Fibonacci sequence. We seek for a closed form, and then we can calculate faster and get more information.

Here, I will show a way based on linear algebra to calculate the closed form of the Fibonacci sequence. In fact, to every linear currence relation, we can get their fomular by the characteristic equation.

After the Fibonacci part, I want to introduce an interesting problem. The Joseph problem. I believe most of us know this problem, so I will state the details briefly. There are n people standing in cycle. From the first one, we start counting from the first person. If someone’s counting number is k, then he would be killed. The problem is ,given a certain n and k, calculating the last person who alive.

We can write a program with using list which has a O(n*k) complexity. In fact, it has a more beautiful way to solve this problem.

We might as well think a brief problem. We make the k=2.

We define a function J. We input N, then it outputs the answer.b
